# ANCAPISTAN

Bienvenido. Esta es mi web personal dedicada a **explicar y difundir el pensamiento libertario y anarcocapitalista** desde mi punto de vista.

## ¿Qué es este proyecto?

Siempre me interesó entender **la libertad individual, el libre mercado y el rol del Estado** en nuestras vidas. En esta web recopilo y organizo ideas clave del pensamiento libertario y anarcocapitalista, explicadas de forma directa y sencilla.

El sitio incluye:

- Definiciones claras de los principios básicos del **pensamiento libertario**: libertad individual, propiedad privada, mercado libre y principio de no agresión.
- Una **selección de libros y autores** que considero fundamentales para profundizar en estas ideas.
- Una sección dedicada a **falacias comunes**, donde analizo críticas habituales y explico por qué muchas parten de supuestos erróneos.
- Contenido adicional como artículos, curiosidades y un chat IA para explorar los temas de forma más interactiva.

Mi intención en esta web no es sentar cátedra ni convencer a nadie a la fuerza, sino **invitar a pensar y cuestionar**. Para martillar ya tengo el substack.

## ¿Por qué lo hice?

No encontré un recurso en español que reuniera estas ideas de forma clara y estructurada, así que decidí construirlo yo mismo.  
Este proyecto es tanto una herramienta de divulgación como una forma de ordenar mi propio aprendizaje.

## Cómo está hecho

- Web estática en HTML/CSS/JavaScript
- Publicada mediante **GitHub Pages**
- Sin backend ni dependencias complejas

## Cómo usar el sitio

- Navega por las secciones desde el menú principal.
- Usa el contenido como punto de partida para investigar por tu cuenta.
- Si detectas errores o tienes sugerencias, puedes abrir un *issue* en el repositorio.

## Estado del proyecto

El proyecto está en evolución. El contenido y la estructura pueden cambiar a medida que se añadan nuevas secciones o se refinen las existentes.
